❓ Quiz Game Screen
Everyone

What it does

The AI (Azure OpenAI / gpt-4.1) has already made questions about your topic. Each question has 4 coloured answer buttons in a 2×2 grid — just like Kahoot!

How to play

  • 1 Read the question at the top
  • 2 Tap one of the 4 coloured buttons — ■ A, ■ B, ■ C, ■ D
  • 3 Instantly see if you got it right!
  • 4 ✅ Correct → 🕺 dance animation, +1 point
  • 5 ❌ Wrong → 💩 poop animation + the right answer is shown
  • 6 Host taps Next Question → to continue

🛠️ How It's Built
⚛️
React Native Web
Makes the screens look great in a browser. All the buttons, animations, and colours.
🔌
Socket.IO
Real-time connection between all players. When you answer, everyone sees it instantly.
🤖
Azure OpenAI (gpt-4.1)
Makes 5–10 quiz questions about any topic you type in — completely automatically!
🟩
Node.js Server
The "brain" behind the game. Tracks rooms, scores, and who answered what.
☁️
Azure Cloud
Hosts the game online so anyone in the world can play — not just on your computer!
📄
Session Logging
Every game is saved to a log file — so we can trace exactly what happened if something goes wrong.

🚀 How to Start the Game (Local)
🖥️ Step 1 — Start the Server
cd C:\Users\prchugh\quizjex\server
node index.js
You should see: 🎮 QuizJex running on http://localhost:3001
🌐 Step 2 — Start the App
Set-Location 'C:\Users\prchugh\quizjex\client'
& ".\node_modules\.bin\expo.ps1" start --web --port 8081
Then open: http://localhost:8081
